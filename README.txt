الإعداد والتثبيت
================

1. متطلبات النظام:
   - PHP 7.4 أو أحدث
   - MySQL 5.7 أو أحدث (أو MariaDB 10.3+)
   - Apache أو Nginx (XAMPP / WAMP / Laragon)

2. تثبيت قاعدة البيانات:
   - افتح phpMyAdmin
   - اختر "New Database" وأنشئ قاعدة بيانات باسم: cafe_management
   - اضغط على "Import" واستورد ملف: install.sql

3. إعداد المشروع:
   - انقل مجلد "cafe" إلى مجلد htdocs في XAMPP
     المسار: C:\xampp\htdocs\cafe
   - أو اضبط Apache ليشير إلى هذا المجلد

4. إعداد قاعدة البيانات في config/database.php:
   - DB_HOST: localhost
   - DB_USER: root
   - DB_PASS: كلمة مرور MySQL (عادة فارغة في XAMPP)
   - DB_NAME: cafe_management
   - BASE_URL: /cafe (غيّره لو المجلد مختلف)

5. إنشاء حساب الأدمن:
   - افتح المتصفح: http://localhost/cafe/setup.php
   - أدخل بيانات المدير وانتهِ من الإعداد
   - احذف ملف setup.php بعد الانتهاء!

6. تسجيل الدخول:
   http://localhost/cafe/login.php

هيكل الصفحات
=============

📊 لوحة تحكم الأدمن (admin/)
  - dashboard    : إحصائيات + خريطة الطاولات الحية
  - categories   : إدارة فئات المنيو
  - items        : إدارة الأصناف والتسعير والكوست
  - tables       : إضافة/تعديل الطاولات
  - employees    : إدارة الموظفين والصلاحيات
  - stock        : إدارة المخزون وحركاته
  - reports      : تقارير يومية وشهرية + رسوم بيانية

💼 لوحة الكاشير (cashier/)
  - index        : تسجيل الحضور (check-in)
  - tables       : عرض الطاولات المتاحة/المشغولة
  - order        : نقطة البيع (إضافة أصناف للطلب)
  - receipt      : طباعة الفاتورة
  - shift        : ملخص الشيفت + تسجيل الانصراف
  - stock        : عرض المخزون (حسب الصلاحية)

======================================================

الصلاحيات المتاحة للكاشير
===========================
- manage_orders  : إدارة الطلبات
- close_orders   : إغلاق الطاولات
- manage_stock   : إدارة المخزون
- view_reports   : عرض التقارير
- apply_discount : تطبيق الخصومات
